## Build distribution

    ./gradlew dist

    
## Install locally

    ./gradlew install
    
## Verify version

    ./gradlew version
    
## Maven POM

    <repository>
      <id>jpos</id>
      <name>jPOS Central Repository</name>
      <url>http://jpos.org/maven</url>
      <layout>default</layout>
    </repository>

    <dependency>
      <groupId>org.jpos</groupId>
      <artifactId>jpos</artifactId>
      <version>1.8.9-SNAPSHOT</version>
    </dependency>
